# Shareholder Finder - Local Web App

This folder contains the **Web Version** of the app, but configured to run on your own computer. This avoids the memory limits of the internet and is much faster.

## How to Install and Run

1.  **Install Python**: Download and install **Python 3.12** from [python.org](https://www.python.org/downloads/). 
    *   **Crucial**: Check the box **"Add Python to PATH"** during installation.

2.  **Setup (One-time)**: Double-click **`Setup_Local.bat`**. This will install all the AI tools onto your computer. It takes about 5 minutes.

3.  **Run the App**: Double-click **`Run_Local_App.bat`**. 
    *   A black window will open.
    *   A few seconds later, a website will automatically open in your browser.
    *   You can now upload your photos and find phone numbers!

## Why use the Local Version?
- **Unlimited Memory**: It won't crash like the Streamlit Cloud version.
- **Privacy**: Your photos stay on your own computer.
- **No Internet Hosting Needed**: You don't need GitHub or Streamlit Cloud.
